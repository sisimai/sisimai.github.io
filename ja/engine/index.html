{{ site.html.ss }}のengine()メソッド

{{ site.html.ss }} v4.16.0からengine()メソッドがSisimaiクラスで 利用可能になりました。下記のサンプルコードのように、 このメソッドは実装されている全てのMTA/MSPモジュールの名前とその説明を一覧として 返す機能を提供します。

Perl

Sisimai->engine()

#!/usr/bin/env perl
require Sisimai;
my $v = Sisimai->engine;
print keys %$v;         # {{ site.sisimai.spec.modules }}
for my $e ( keys %$v ) {
    print $e;           # Exchange
    print $v->{ $e };   # Microsoft Exchange Server
}
Ruby

Sisimai.engine()

#!/usr/bin/env ruby
require 'sisimai'
v = Sisimai.engine
puts v.keys.size;  # {{ site.sisimai.spec.modules }}
v.keys.each do |e|
  puts e    # Exchange
  puts v[e] # Microsoft Exchange Server
end
