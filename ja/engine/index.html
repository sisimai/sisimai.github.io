{{ site.html.ss }}のengine()メソッド

{{ site.html.ss }} v4.16.0からengine()メソッドがSisimaiクラスで 利用可能になりました。下記のサンプルコードのように、 このメソッドは実装されている全てのMTAモジュールの名前とその説明を一覧として 返す機能を提供します。

Perl

Sisimai->engine()

#!/usr/bin/env perl
require Sisimai;
my $v = Sisimai->engine;
print keys %$v;         # {{ site.sisimai.spec.modules }}
for my $e ( keys %$v ) {
    print $e;           # Sisimai::Bite::Email::Exchange2007
    print $v->{ $e };   # Microsoft Exchange Server 2007
}
Ruby

Sisimai.engine()

#!/usr/bin/env ruby
require 'sisimai'
v = Sisimai.engine
puts v.keys.size;  # {{ site.sisimai.spec.modules }}
v.keys.each do |e|
  puts e    # Sisimai::Bite::Email::Exchange2007
  puts v[e] # Microsoft Exchange Server 2007
end
