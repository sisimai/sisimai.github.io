動作環境

{{ site.html.ss }}(シシマイ)は*BSDやMac OS XやLinuxのようなPerlまたは Ruby(JRubyを含む)が動作するOSで動きます。また、{{ site.html.ss }} 4.22.5からは、 Windows上のActivePerlで動作するかもしれません。

Perl {{ site.sisimai.lang.perl.min }}以上

  • 2つの依存モジュール
    • Class::Accessor::Lite
    • JSON
  • 幾つかのコアモジュール
    • Module::Load
    • Time::Piece
    • Time::Local
CPANからダウンロード

Ruby {{ site.sisimai.lang.ruby.min }}以上

  • JRuby 9.0.4.0以上でも可
  • 依存ライブラリ(MRI)
    • Oj
  • 依存ライブラリ(JRuby)
    • JrJackson
RubyGemsからダウンロード

{{ site.html.ss }}をインストールする

{{ site.html.ss }}はGitHub, CPAN, RubyGems.orgからそれぞれダウンロード・インストールが可能です。 また、Perl版{{ site.html.ss }}はFreeBSDのportsコレクションからでもインストールすることができます。

GitHubから

開発版を含む最新バージョンと安定版の{{ site.html.ss }}は GitHub のmasterブランチからダウンロード出来ます。Perl版・Ruby版ともに このリポジトリにあります。

Perl

Perl版{{ site.html.ss }}

Perl版{{ site.html.ss }}のリポジトリ名は sisimai/p5-Sisimai です。 

% git clone https://github.com/sisimai/p5-Sisimai.git
% cd ./p5-Sisimai
% make install-from-local
--> Working on .
Configuring Sisimai-4.20.0 ... OK
1 distribution installed

Ruby版{{ site.html.ss }}

Ruby版{{ site.html.ss }}のリポジトリ名は sisimai/rb-Sisimai です。

Portsコレクションから(FreeBSD)

Perl版{{ site.html.ss }}はFreeBSD のPortsコレクション (mail/p5-Sisimai) インストールができます。makeコマンド一発でインストールが可能ですが、 Perl版{{ site.html.ss }}がPortsコレクションに取り込まれるタイミングによっては、 最新版ではない{{ site.html.ss }}がインストールされる点に注意してください。

基本的な使い方

make()

{{ site.html.ss }}のmake()メソッド

下記のコードは{{ site.html.ss }}のmake() メソッドを使ってバウンスメールを解析し解析結果のデータを得る例です。 make()メソッドに指定できる他の引数については シシマイでの解析方法をご覧ください。

Perl

Perl版{{ site.html.ss }}

#! /usr/bin/env perl
use Sisimai;
my $v = Sisimai->make('/path/to/mbox'); # or Path to Maildir

if( defined $v ) {
    for my $e ( @$v ) {
        print $e->addresser->address;   # shironeko@example.org # From
        print $e->recipient->address;   # kijitora@example.jp   # To
        print $e->recipient->host;      # example.jp
        print $e->deliverystatus;       # 5.1.1
        print $e->replycode;            # 550
        print $e->reason;               # userunknown
    }
} else {
    # There is no bounce message in the mailbox
    # or Sisimai could not parse
}
Ruby

Ruby版{{ site.html.ss }}

#! /usr/bin/env ruby
require 'sisimai'
v = Sisimai.make('/path/to/mbox') # or Path to Maildir

if v.is_a? Array
  v.each do |e|
    puts e.addresser.address    # shironeko@example.org # From
    puts e.recipient.address    # kijitora@example.jp   # To
    puts e.recipient.host       # example.jp
    puts e.deliverystatus       # 5.1.1
    puts e.replycode            # 550
    puts e.reason               # userunknown
  end
else
  # There is no bounce message in the mailbox
  # or Sisimai could not parse
end
dump()

{{ site.html.ss }}のdump()メソッド

Perl版{{ site.html.ss }} v4.1.27から登場したdump()メソッドは解析結果を JSONで得るのに便利です。下記のように解析したいメールのPATHを渡すだけの 簡単なコードで済みます。

Perl

Perl版{{ site.html.ss }}

#! /usr/bin/env perl
use Sisimai;
print Sisimai->dump('~/Maildir/cur');
Ruby

Ruby版{{ site.html.ss }}

#! /usr/bin/env ruby
require 'sisimai'
puts Sisimai.dump('~/Maildir/cur')
Demo

dumpメソッドの使用例

jq

"jq"コマンドと組み合わせる

解析結果のJSONデータを目で確認したい場合は jqコマンドが便利です。 下記のようにdump()メソッドの出力が読みやすく表示されます。

Perl

Perl版{{ site.html.ss }}

% perl -MSisimai -lE 'print Sisimai->dump(shift)' path/to/mbox | jq .
Ruby

Ruby版{{ site.html.ss }}

% ruby -rsisimai -e 'puts Sisimai.dump($*.shift)' path/to/mbox | jq .
