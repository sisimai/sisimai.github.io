{{ site.html.ss }}(シシマイ)とは?

{{ site.html.ss }}(シシマイ)は複雑で多種多様なバウンスメールを解析してバウンスした 理由や宛先メールアドレスなど、配信が失敗した結果を構造化データで出力するライブラリで JSONでの出力も可能です

現在の安定版最新バージョンは2月2日にリリースした{{ site.html.ss }} {{ site.sisimai.version }}で GitHub(Perl版)または GitHub(Ruby版)からのみ 入手可能です。{{ site.html.ss }} {{ site.sisimai.version }}の変更点は リリースノート を見てください。 {{ site.html.ss }} 5で入った非互換な変更点はSisimai 4との違いを 確認してください。 現時点でCPANおよび RubyGems.orgから ダウンロードできるのはv4.25.16です。 2024年2月29日(木)にCPANとRubyGemsのパッケージはv5.0.0になります。

解析ライブラリ

{{ site.html.ss }}はUNIX mbox形式のファイルや Maildir/ディレクトリ内にあるような バウンスメール用解析ライブラリです。PerlまたはRubyで動作します。

vs. {{ site.html.bh }}

2014年夏、Perl版{{ site.html.ss }}(シシマイ)は {{ site.html.bh }}が抱える解析精度に関わる致命的な問題の数々 (一通のメールに複数件のバウンスがある場合に一件目しか解析出来ない、依存モジュールが多く、 うち幾つかは既にメンテナンスされていない、Perl 5.16以降で動作しない等) を解決する為に開発されました。

現在、Perl版{{ site.html.ss }}は{{ site.html.bh }}の{{ site.sisimai.diff.timesfaster.perl }} 倍程度高速に動作し、2倍の解析精度を誇ります。そして2016年2月3日にはRuby版{{ site.html.ss }} をリリースしました。

そして2024年2月2日、{{ site.html.ss }}公開から10周年の年に、ドメイン認証 (SPF, DKIM, DMARC, ARC)系エラーやレピュテーションに起因するエラーなど 現代のバウンスも検出可能な{{ site.html.ss }} {{ site.sisimai.version }} をリリースしました。

{{ site.html.ss }}(シシマイ)は*BSDやMac OS XやLinuxのようなPerlまたは Rubyが動作するOSで動きます。また、{{ site.html.ss }} 4.22.5からは、 Windows上のActivePerlで動作するかもしれません。

サポート

{{ site.html.ss }}はオープンソースライセンス(二条項BSD) で公開しているライブラリですので、どなた様でも自由にお使い頂けます。
しかし、バウンスメール解析の為の環境構築や {{ site.html.ss }}周辺スクリプトの開発、解析済データを蓄積するデータベース設計や管理、 継続的なバウンス管理、そしてこれまでのバウンスメール解析システムである {{ site.html.bh }}からの移行などをサポートする品目を、 開発元である株式会社{{ site.html.cr }}にて提供しております。
