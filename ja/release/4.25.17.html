{{ site.html.ss }} 4.25.17

2025/03/12

{{ site.html.ss }} 4.25.17をリリースしました。 このバージョンにおける変更点は、 3月13日以降でNTTドコモ宛のメールでドメインして拒否や 指定受信によるバウンスメールが返ってこなくなる(迷惑メールフォルダに入る)変更 への対応のみです。

{{ site.html.ss }} 4系は全体がクラッシュするような致命的バグの修正と 影響の大きなサービスの変更に対応するのみの状態になっていますので、ドメイン認証やレピュテーションなど、 現代の配信エラーに対応した{{ site.html.ss }} 5系へのアップグレードをおすすめします。

{{ site.html.ss }} 5.0.0でVersion 4系とは非互換な変更が多く入っていますので、 インストールまたはアップグレードの前に実行環境のPerlやRubyのバージョンを確認してください。

変更点

  • NTTドコモから返ってきたバウンスメールのエラーメッセージに 550 Unknown user と書いている場合、検出されるバウンス理由は User Unknown のみになります。 #577 #335
