Release Notes

{{ site.html.ss }} 5.0.1

2024/03/03

{{ site.html.ss }} 5.0.1をリリースしました。Perl版{{ site.html.ss }}に特有の変更は Perlが、Ruby版{{ site.html.ss }}に特有の変更は Rubyが、それぞれ冒頭に入っています。 {{ site.html.ss }} 5.0.0でVersion 4系とは非互換な変更が多く入っていますので、 インストールまたはアップグレードの前に実行環境のPerlやRubyのバージョンを確認してください。

変更点

  • Perl t/023-mail-stdin.t でテストが落ちるバグを修正 78465c7
  • Perl t/500-fact.tと t/600-lhost-code.t でテストが落ちるバグ(use YAML;)を修正 (@eserteさんからの報告: #505)
  • Simple Mail Transfer Protocol (SMTP) Enhanced Status Codes Registry で定義される三つのSMTPステータスコード: 5.7.28と5.7.29と5.7.28 をSisimai::SMTP::Statusクラスに実装 #501 #263
  • lhost-postfix-28.emlと lhost-postfix-28.emlの バウンス理由をNotCompliantRFCに修正 #502 #264
  • GoogleのGmail SMTP errors and codes およびEmail sender guidelines で言及されているSMTPエラーコードをSisimai::Rhost::Googleクラスに追加実装 #504 #265
