{{ site.html.ss }} 5.2.1

2025/03/12

{{ site.html.ss }} 5.2.1をリリースしました。 このバージョンにおける主な変更点は、 3月13日以降でNTTドコモ宛のメールでドメインして拒否や 指定受信によるバウンスメールが返ってこなくなる(迷惑メールフォルダに入る)変更 への対応と Cloudflare Email Routing の返すバウンスメールへの部分的な対応などです。

Perl版{{ site.html.ss }}に特有の変更はPerlが、 Ruby版{{ site.html.ss }}に特有の変更はRubyが、 Go版{{ site.html.ss }}に特有の変更はGoが、 それぞれ冒頭に入っています。

{{ site.html.ss }} 5.0.0でVersion 4系とは非互換な変更が多く入っていますので、 インストールまたはアップグレードの前に実行環境のPerlやRubyのバージョンを確認してください。

変更点

  • NTTドコモから返ってきたバウンスメールのエラーメッセージに 550 Unknown user と書いている場合、検出されるバウンス理由は User Unknown のみになります。 #573 #333 #64
  • Cloudflare Email Routing が返す一部のエラーコードに対応したSisimai::Rhost::Cloudflareを実装しました。 #534 #336 #66
  • Perl Sisimai::SMTP::Statusでsubstr関数が範囲外の 文字を参照するバグを修正しました
    Thanks to @ViktorNacht #574 #575
  • Go 名前が紛らわしいのでsisimai/stringパッケージは sisimai/mojiに変更されました #68 #69
