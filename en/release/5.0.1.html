Release Notes

{{ site.html.ss }} 5.0.1

2024/03/03

Sisimai 5.0.1 has been released! The main changes are the implementation of the newly released SMTP error codes in Gmail and the bug fixes in the test code for the Perl version. Changes specific to the Perl version of {{ site.html.ss }} are indicated by Perl at the beginning of each item, and changes specific to the Ruby version of {{ site.html.ss }} are indicated by Ruby. And then, {{ site.html.ss }} 5 includes many changes that are not compatible with Version 4, so please check the Perl or Ruby version of your environment before installing or upgrading.

Changes

  • Perl Fixed a bug that caused tests to fail in t/023-mail-stdin.t. 78465c7
  • Perl Fixed a bug (use YAML;) that caused tests to fail in both t/500-fact.t and t/600-lhost-code.t. Thanks to @eserte #505
  • Implement three SMTP status codes: 5.7.28, 5.7.29, and 5.7.28 listed in Simple Mail Transfer Protocol (SMTP) Enhanced Status Codes Registry at Sisimai::SMTP::Status class. #501 #263
  • Fixed the bounce reason in the decoded results of lhost-postfix-28.eml and lhost-postfix-28.eml to NotCompliantRFC. #502 #264
  • Implemented additional SMTP error codes and messages listed in Google's Gmail SMTP errors and codes and Email sender guidelines to the Sisimai::Rhost::Google class. #504 #265
