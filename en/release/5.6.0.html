{{ site.html.ss }} 5.6.0

2026/02/02

{{ site.html.ss }} 5.6.0 has been released. The main changes in this version include: the consolidation of four bounce reasons into two, refining error message patterns for better bounce classification, and restructuring the internal status code system.

Changes in the Perl version of {{ site.html.ss }} are indicated by Perl at the beginning of each item, changes in the Ruby version of {{ site.html.ss }} are indicated by Ruby, and changes in the Go version of {{ site.html.ss }} are indicated by Go.

And then, {{ site.html.ss }} 5 includes many changes that are not compatible with Version 4, so please check the Perl or Ruby version of your environment before installing or upgrading.

  • Go Perl Ruby Consolidate four bounce reasons into two: #307 #634 #399
    • Merge ExceedLimit and MesgToobig into EmailTooLarge .
    • Merge Speeding and TooManyConn into RateLimited .
  • Go Perl Ruby Reorganize the internal status code for assigning codes when no DSN value is available in a bounce email. #322 #638 #403
