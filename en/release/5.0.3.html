Release Notes

{{ site.html.ss }} 5.0.3

2024/05/22

{{ site.html.ss }} 5.0.3 has been released! The main changes in this release are the following updates of Gmail SMTP errors and codes on April 30 and May 10, following SMTP error codes and error messages returned from Exchange Online, and 16 new error message patterns.

Changes specific to the Perl version of {{ site.html.ss }} are indicated by Perl at the beginning of each item, and changes specific to the Ruby version of {{ site.html.ss }} are indicated by Ruby.

And then, {{ site.html.ss }} 5 includes many changes that are not compatible with Version 4, so please check the Perl or Ruby version of your environment before installing or upgrading.

Changes

  • Perl Declare use v5.26; at each file using the postfix dereference #511 #512
    • Thanks to CPAN Testers: Slaven Rezić, David Cantrell, Chris Williams, and Andreas J. König
  • Remove .travis.yml from this repository (Migrated to GitHub Actions)
  • Follow updates in Gmail SMTP errors and codes on Apr 29 and May 10 at Sisimai::Rhost::Google #513 #515 #519 #275 #277 #283
    • The following SMTP error codes implementd 4.7.23 4.7.30 4.7.32 5.7.29 5.7.30

    • github.com/azumakuniyuki/feb-2024-no-auth-no-entry@1d6adede
    • github.com/azumakuniyuki/feb-2024-no-auth-no-entry@d477b178
