What is {{ site.html.ss }} ?

{{ site.html.ss }} is the successor of {{ site.html.bh }}, is a Mail Analyzing Interface: a Perl module and a Ruby Gem for parsing RFC5322 bounce emails and for generating structured data such as JSON.

{{ site.html.ss }} {{ site.sisimai.version }} was released on {{ site.sisimai.release.en }}. This release include a bug fix for a serious problem (only in the Perl version of Sisimai) and other tiny updates for DNS related error message patterns. v4.25.10 is the final release of the version 4 series. However, we will release v4.25.11 if a serious bug were found after today. And then, Sisimai 5 and the Go language version of Sisimai are under development on sisimai/go-sisimai repository and will be released until the next spring or summer. For further information about all the changes on this version available at blog.libsisimai.org.

Parse Bounce Emails

{{ site.html.ss }} is a library to parse bounce emails in a UNIX mbox file or emails in the Maildir/ directory.

Details

vs. bounceHammer

On August 2014, We have started to develop a Perl version of {{ site.html.ss }} to resolve many fundamental problems of {{ site.html.bh }} which are related to analytical precision (it could not be detected 2 or more bounces in a single email), many dependencies of external modules (some modules have not maintained for a long time), and cannot work on Perl 5.16 or later.

As of the present, the Perl version {{ site.html.ss }} is about 2.2 times faster than bounceHammer, have 2.0 times higher of analytical precision, depends only 2 external modules, and can work on Perl {{ site.sisimai.workson.perl[0] }} or later versions of Perl. From August 2015, We have started to develop a Ruby version of {{ site.html.ss }}. The software has been released on Feburary 3, 2016.
